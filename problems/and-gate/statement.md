# AND Gate

Implement a 2-input AND gate.

## Ports

| Name | Direction | Width | Description |
|------|-----------|-------|-------------|
| `a`  | input     | 1     | First operand |
| `b`  | input     | 1     | Second operand |
| `y`  | output    | 1     | `a AND b` |

## Behavior

`y` must be `1` only when both `a` and `b` are `1`; otherwise `y` must be `0`.
